# HexinWallpaper - 壁纸小程序（uni-app）

基于 B 站 UP 主“咸虾米”的《虾米壁纸》项目改良而来的壁纸类微信小程序。本项目使用 uni-app 开发，可作为 uni-app 学习参考或课程作业。

## 📱 项目简介

HexinWallpaper 是一个壁纸展示与预览的小程序，目前已集成约 60 张壁纸图片，支持壁纸列表浏览、点击查看大图等基础功能，（您可以下载下来进行预览）。项目整体框架完整，可用于提交期末课程设计或作为 uni-app 入门练习的示例。

> 由于个人学业安排（即将大三，准备考研），该项目后续可能不再进行大规模维护，但现有功能已满足基本演示需求。若您有兴趣，欢迎在此基础上继续完善。

## ✅ 已实现功能

- 壁纸列表展示（分类或瀑布流布局）
- 壁纸详情预览（点击图片进入大图查看）


## 🚧 待完成 / 未完善功能

1. **搜索功能**：搜索框目前仅有弹窗提示，实际搜索跳转逻辑尚未实现。
2. **个人中心**：
   - “我的更新” 仅弹窗提示，未接入实际数据。
   - “我的反馈” 仅弹窗提示，未实现提交反馈逻辑。
   - 搜索框 UI 展示（弹窗提示，未跳转）
   - 个人中心页面（部分功能弹窗提示）
3. **图片预览 BUG**：部分壁纸在列表页点击后，进入的详情页图片与预期不符（图片对应关系错乱），多次尝试修复未果，问题根源尚未定位。

## 🐛 已知问题

- **预览页与列表图片不一致**：部分壁纸点击后显示的图片与列表缩略图不匹配，疑似数据索引或路由传参错误。曾尝试用 AI 修复，但效果不佳，属于“拆东墙补西墙”的临时方案。如果您能定位问题，欢迎提交 PR 或 Issue。

## 📂 实验内容（1-9）

本项目包含 `experiment` 目录，内含 **uni-app 学习实验 1~9**，每个实验均配有：

这些实验是学习 uni-app 开发过程中的练习，涵盖基础组件、API 调用、页面路由等知识点。  
**适用场景**：若您所在学校的 uni-app 实验内容相似，可直接复用；若不同，可参考思路并做合理修改。

## 🛠️ 运行方式

1. 使用 **HBuilderX**（推荐）导入本项目。
2. 选择 `uni-app` 运行环境（微信小程序 / H5 / App）。
3. 编译并运行至对应平台即可预览。
